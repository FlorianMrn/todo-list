Todo list
====================

Purpose
--------------------

The famous todo list, to continue to evolve.

Tools used
--------------------

- Backend: Node.js, Express.js, Mongoose
- Database: MongoDB
- Frontend: React, hooks

How to run app locally
--------------------

- Frontend
  * `cd front/`
  * `yarn`
  * `yarn start`

- Backend
  * `cd back/`
  * `yarn`
  * `yarn nodemon` (to use nodemon)
  
Language
